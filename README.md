# ReadersHaven
Author: De'ovyun LeRouge
Overview

The online application called Reader's Haven was created to assist users in finding new books based on their search terms or mood. Users may keep and manage their reading list in one location using the program, which also offers tailored book recommendations.
This project's objective was to establish a user-friendly, aesthetically pleasing platform that blends functional web development with user-centered design. Users can use direct search or mood-based discovery to browse books, record their choices, and monitor their reading progress.

Purpose

Reader's Haven aims to make the process of finding books easier and more interesting. Finding books that fit their interests or mood is a challenge for many readers. This program solves that issue by offering mood-based recommendations, providing keyword searches, and allowing users to save and manage books in a personalized reading list.

Live Site

https://dlerouge.github.io/ReadersHaven/list.html

Features

    1. Search books by keywords
    2. Discover books by mood (Cozy, Romantic, Dark Academia, etc.)
    3. Save books to a reading list
    4. Track reading status (Want to Read, Started, Finished)
    5. Responsive layout for different screen sizes.

Technology Used
    1. HTML5
    2. CSS3
    3. JavaScript
    4. Google Books API
    5. Supabase
